# 社員紹介動画の質問項目 設計ガイド 動画スクリプト案

## 動画のトーンと対象
- ターゲット: 採用担当者、HR部門のリーダー、インタビューを企画するディレクター
- トーン: 実践的、プロフェッショナル、ノウハウ共有型
- 動画の長さ: 約2分半〜3分を想定

---

## スクリプト構成

### 1. イントロダクション（0:00 - 0:30）
- [画面] 「社員紹介動画の質問項目」というタイトルとともに、採用動画の制作で悩む担当者のイメージイラスト。
- [ナレーション] 「社員紹介動画を作ろうとしたとき、どんな質問を用意すべきか迷った経験はありませんか？『業務内容』や『社風』といった定番のテーマだけでは、求職者の記憶に残る動画は作れません。」
- [画面] AI動画生成のパイプラインや「AI」の文字が光る演出。
- [ナレーション] 「今回は、求職者の心を動かし、さらに『AI動画生成ツール』にも流し込みやすいインタビューの構成設計について解説します。」

### 2. 「何を聞くか」より「なぜ聞くか」（0:30 - 1:00）
- [画面] 単語「やりがい」「社風」に×印。具体的なエピソードの吹き出しに○印。
- [ナレーション] 「求職者が求めているのは、企業の公式メッセージではなく『リアルな実感』です。『当社の魅力を教えてください』という直接的な質問は、当たり障りのない回答を生みがちです。そうではなく、『入社して最初の1か月で一番驚いたことは？』のように、具体的なエピソードを自然に引き出す誘導路を作ることが重要です。」

### 3. AIに適したインタビュー設計とは（1:00 - 1:45）
- [画面] インタビュー音声がテキスト化され、AIが動画を自動生成していくフロー図。
- [ナレーション] 「また、近年では撮影した素材をAIツールでスクリプト化・動画化する制作フローが主流になりつつあります。このとき、AIの処理精度を上げるための3つの原則があります。」
- [画面] 「1回答1トピック」「STARフォーマット」「事前ブリーフィング」の3つのキーワードテロップ。
- [ナレーション] 「一問に複数のテーマを入れず、1回答1トピックを徹底すること。そして状況・課題・行動・結果の順で話す『STARフォーマット』へ誘導すること。最後に、収録前のブリーフィングで具体的な事実や数値を意識して話してもらうよう伝えることです。」

### 4. ターゲット別質問カスタマイズ（1:45 - 2:20）
- [画面] 「若手層」と「即戦力層」の分岐点を示すアニメーション。それぞれの質問例を対比。
- [ナレーション] 「採用ターゲットによる質問のカスタマイズも不可欠です。若手層には『失敗しても挑戦できる環境か』、つまり入社後の初仕事やサポートについての質問を。一方、即戦力層には『意思決定のスピード』や『自分の専門性が活かされる場面』といった、活躍機会に重点を置いた質問が効果的です。」

### 5. クロージング（2:20 - 2:40）
- [画面] リスト化された質問項目を整理しているHR担当者の実写またはイラスト。
- [ナレーション] 「良い動画は、良い質問設計から生まれます。まずは現在の質問項目リストを見直し、具体的なエピソードを引き出せる質問へ書き換えてみましょう。本記事ではより詳しい質問例やカテゴリごとの解説を公開しています。ぜひチェックしてみてください。」
- [画面] 記事へのリンクやロゴを表示してフェードアウト。
