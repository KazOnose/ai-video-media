●シーン3：具体的メリット（30-50秒）
- 映像イメージ：[MOGRT: B2B_3点メリット] を使用。3つのポイントを表示。
- ナレーション：ターゲット設計の効果は３つ。１：求職者の課題に深く共感できる。２：スキルの活かし方が具体的に伝わる。３：本当に欲しい人材とのマッチング精度が劇的に向上する。

●シーン4：クロージング（50-60秒）
- 映像イメージ：サービスロゴと公式URL。
