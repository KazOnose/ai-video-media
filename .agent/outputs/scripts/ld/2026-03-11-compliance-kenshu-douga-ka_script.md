# コンプライアンス研修を動画化して「伝わるコンテンツ」に変えるポイント

## 動画のトーンと対象
- ターゲット: L&D（教育・研修）担当者、人事担当者、コンプライアンス責任者
- トーン: 実践的、教育的、共感を生むノウハウ系の解説
- 動画の長さ: 約2分〜2分半を想定

---

## スクリプト構成

### 1. イントロダクション（0:00 - 0:25）
- [画面] 「またコンプライアンス研修か…」と頭を抱える社員と、悩む人事担当者のイラスト。
- [ナレーション] 「『また同じ内容だ』『テキストを読んだだけ』…毎年のコンプライアンス研修で、こんな受講者の声に悩んでいませんか？重要なルールが確実に定着せず、形骸化してしまう問題。この『伝わらない』壁を突破するカギは、研修の『動画化』にあります。」

### 2. なぜ「ドラマ」と「図解」が効果的なのか（0:25 - 1:05）
- [画面] 抽象的な法律の条文（テキスト）が、ドラマ風のシーン（上司と部下の会話）に切り替わるアニメーション。
- [ナレーション] 「コンプライアンスの規程は抽象的で、そのままでは記憶に残りません。しかし、物語の力によって『自分ごと化』させるドラマ仕立ての動画なら、『自分がこの状況に置かれたらどうするか』という実務に直結した判断力を育てることができます。特にグレーゾーンの判断訓練に効果的です。」
- [画面] 複雑な報告フロー（情報漏洩など）が、一つ一つの矢印と吹き出しで図解されていくアニメーション。
- [ナレーション] 「また、『なぜそのルールが必要か』『違反するとどうなるか』という因果関係を伝えるには、段階的に表示される図解動画が最適です。仕組みを視覚的に理解することで、ルール遵守の動機付けが強化されます。」

### 3. おすすめの「4段構成」（1:05 - 1:40）
- [画面] 4つのステップを表すフローテロップ：「問題提起」→「ドラマ」→「解説」→「チェック」
- [ナレーション] 「動画を制作する際に最もおすすめなのが、『問題提起』で共感を呼び、『ドラマ』でリアルな状況を見せ、『解説』で図解等を使ってルールを整理し、最後に『チェックテスト』で確認する4段構成です。」
- [画面] 時計のアイコンと「1テーマ5〜8分」という文字。
- [ナレーション] 「これらを1つのテーマにつき5〜8分程度の動画に分割することで、受講者の集中力を持続させ、テーマごとの更新・管理も容易になります。」

### 4. コストを抑える運用のコツ（1:40 - 2:05）
- [画面] 「シナリオテンプレート」「社員の協力（スマホ撮影など）」のアイコン。
- [ナレーション] 「『ドラマ仕立てはコストがかかる』と思われがちですが、社員が日常の職場で演じる『5分以内のシナリオ動画』であれば、スマホ撮影でも十分に効果を生みます。よくあるシーンのテンプレートを作っておくことで、内製化のハードルはぐっと下がります。」

### 5. クロージング（2:05 - 2:20）
- [画面] 笑顔で研修動画を受講している社員の姿。
- [ナレーション] 「コンプライアンス研修の動画化は、組織に実践的なルールを定着させる具体的な手段です。まずは理解度が浅い1テーマから動画への改善を始めてみませんか？詳細なノウハウは本記事をご覧ください。」
- [画面] チャンネル登録・記事リンクの案内をしてフェードアウト。
