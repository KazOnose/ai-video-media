# 動画台本｜チャットボット動画連携

【ターゲットキーワード：チャットボット 動画 連携】

●シーン1：フック（0-10秒）
- 映像イメージ：[MOGRT: B2B_結論強調_横型] を使用。チャットウィンドウにテキストが流れ込んでくる映像の後、「解決できない問い合わせ」を示す赤いフォーカス演出。
- ナレーション：「ボットの回答がテキストだけで、顧客が結局また電話してくる。そのギャップ、動画で埋められます。」

●シーン2：ソリューション提示（10-30秒）
- 映像イメージ：[MOGRT: B2B_操作説明サイドバー_横型] を使用。チャットボットの管理画面で回答テンプレートに動画URLを追加するUI操作のイメージ。
- ナレーション：「チャットボットの回答に動画URLを差し込むことで、テキストでは伝えにくい操作手順や設定変更を視覚的に案内できます。Zendeskなどのプラットフォームなら、管理画面から設定するだけで対応可能です。」

●シーン3：具体的メリット（30-50秒）
- 映像イメージ：[MOGRT: B2B_3点メリット] を使用。3つのKPI改善ポイントをアイコンとテキストで表示。
- ナレーション：「メリットは三つ。第一に自己解決率の向上。第二に有人対応へのエスカレーション件数の削減。第三に顧客満足度スコア（CSAT）の改善。動画と連携することで、ボットの回答品質を一段引き上げられます。」

●シーン4：クロージング（50-60秒）
- 映像イメージ：AI Strategic Insight のロゴと公式URL。
- ナレーション：「サポート体制の自動化と顧客体験の向上を同時に実現したい企業は、AI Strategic Insight にご相談ください。」
